SuperSTore Sales Insight Generation

In the Superstore Sales Analysis project, I cleaned and transformed raw data by removing duplicates, handling missing values, and creating calculated fields. Using Power BI, I modeled the data with a star schema, linking dimension tables to a central fact table and creating DAX measures for key metrics. I developed interactive dashboards showcasing KPIs, regional performance, sales trends, and diagnostic insights like loss-making products and return rates, enabling stakeholders to make data-driven decisions across marketing, operations, and logistics.

Student Data Optimization & Monitoring

To optimize training data for analytical modeling, I conducted a thorough review of the ingested dataset, resolving missing values, removing duplicates, standardizing formats, and converting data types to ensure consistency. I engineered new features such as training periods, data quality flags, and category indicators, and used conditional logic to flag outliers and incomplete records, ensuring only high-quality data entered the star schema. Building on this clean foundation, I developed an interactive dashboard showcasing KPIs like total applicants, active periods, top courses, availability types, and career goals. Visuals included KPI cards, bar charts, treemaps for skill distribution, and pie charts for availability, enabling actionable insights and informed training program decisions.
